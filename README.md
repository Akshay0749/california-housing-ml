# California Housing Price Prediction 🏠

An end-to-end Machine Learning project based on Chapter 2 of
**Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow**
by Aurélien Géron.

## 📌 Problem Statement
Predict the **median house value** of California districts
using census data.

## 📊 Dataset
- **Source:** California Housing Dataset (1990 Census)
- **Size:** 20,640 districts × 10 features
- **Target:** `median_house_value`
